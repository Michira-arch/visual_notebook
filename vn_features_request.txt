## Visual Notebook Feature Roadmap (Agent Suggestions)

To improve the integration between the AI assistant and the workspace, the following features are proposed:

### 1. Workspace & UI
- **Sidecar Mode**: Implement an adaptable layout that resizes the notebook workspace when the agent modal is active, preventing occlusion.
- **Rich Component Rendering**: Fix markup rendering in the agent window to support tables, charts, and LaTeX.

### 2. Execution & Reliability
- **Execution Sandbox**: A background environment for the agent to verify code/Mermaid syntax before outputting.
- **Auto-Run Actions**: Ensure that `create_cell` actions for Canvas and Floodlight trigger execution immediately upon user approval, bypassing the "pending" state.

### 3. Intelligence & Context
- **Visual Perception**: Enable the agent to "see" through screenshot attachements. The agent may take a screenshot of current viewport by itself, which would auto-attach, and send it back to the agent for analysis. The agent should also be able to receive images from the user.
